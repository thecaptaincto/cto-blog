# Voices of the World - English Accents Blog

A beautifully designed blog exploring English accents from around the world, featuring a black and white East Asian ink brush aesthetic.

## What's Included

This website contains:

- **1 Homepage** (`index.html`) - Main navigation page with elegant design
- **9 Detailed Blog Posts** - Each covering different regions and their English accents
- **1 Stylesheet** (`style.css`) - All the styling and animations
- **2 Guide Files** - This README and a formatting guide

## The Blog Posts

### 1. Native English Accents (`native-accents.html`)
Explores British, American, Canadian, Australian, Irish, and New Zealand English varieties.

### 2. Nordic & Germanic Excellence (`nordic-accents.html`)

## Design Features

### Visual Style
- **Black and white color scheme** with subtle grays
- **East Asian ink brush aesthetic** with hand-drawn elements
- **Minimalist and elegant** - not overly specific, works for any blog topic
- Ink splash effects, brush strokes, and clean lines
- Smooth animations and hover effects

### Typography
- **Body text:** Crimson Text (elegant serif font)
- **Headers:** Noto Serif SC (Chinese serif - ties to the aesthetic)
- **Drop cap** on first paragraph of each article
- Clean, readable font sizes

### Responsive Design
- Works perfectly on desktop, tablet, and mobile
- Adapts layout for different screen sizes
- Touch-friendly navigation

## How to Use This Website

### Viewing Locally
1. Download all files to a folder on your computer
2. Keep all files in the same folder (don't separate them)
3. Double-click `index.html` to open in your web browser
4. Navigate using the numbered cards on the homepage

### File Structure

## Customizing the Design

### Colors
Edit these variables in `style.css` (lines 3-10):
```css
:root {
    --ink-black: #1a1a1a;      /* Main text color */
    --soft-black: #2d2d2d;      /* Secondary black */
    --charcoal: #4a4a4a;        /* Gray text */
    --paper-white: #fafaf8;     /* Background */
    --off-white: #f5f5f3;       /* Secondary background */
    --ink-grey: #888;           /* Light gray */
    --accent-grey: #d4d4d4;     /* Very light gray */
}
```

### Fonts
To change fonts, edit the `@import` line at the top of `style.css` and update the `font-family` properties.
